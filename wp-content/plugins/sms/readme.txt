== Description ==
Embed sms sending widget in your site and allow your visitors to send free SMS.

Why SMS ?

* Allow your vistors to send free SMS.
* Decrease bounce rate of your site.
* You will have more returing visitors.
* You can send your site url in every SMS sent. This will help you to attract new visitors.
* Upto 50% less than other SMS provider, no matter which package you purchase. It cost just 0.005$ (half of the cent) per sms. 
* No monthly time bound. No setup fee.
* Quick SMS delivery
* Support 200+ countries.

You need API key to use this plugin. You can select your package or get Free Test API key from <a href="http://dnesscarkey.com/sms_service/api/" target="_blank">here</a>

== Installation ==

1. Upload the plugin sms files to the `/wp-content/plugins/` directory
1. Activate the sms plugin through the 'Plugins' menu in WordPress.
1. Select you package, get the API key and verify it from Settings > SMS.
1. Add Footer Text to be send in every SMS.
1. Drag and Drop SMS widget to you sidebar from Apperance > Widgets OR can use [SMS] shortcode to keep in post and page.
1. You are ready to allow you site vistors to send Free SMS
